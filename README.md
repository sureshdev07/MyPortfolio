# Portfolio Website - Suresh Kumar Vijayan

A modern, responsive portfolio website showcasing expertise as a CAD Automation Developer with 6+ years of experience in CAD/CAE tools customization and engineering automation.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that works seamlessly across all devices
- **Dark/Light Mode**: Toggle between dark and light themes with persistent user preference
- **Smooth Animations**: Elegant scroll animations and interactive elements
- **Professional Sections**:
  - Hero section with animated introduction
  - About section with achievements and awards
  - Technical skills showcase
  - Interactive experience timeline
  - Featured projects gallery
  - Contact information

## 🚀 Getting Started

### Local Development

1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website should load with all features working

### Live Server (Optional)

For a better development experience, you can use a live server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (install http-server globally first)
npm install -g http-server
http-server
```

## 🌐 Deployment Options

### GitHub Pages

1. Create a GitHub repository
2. Push all files to the repository
3. Go to Settings → Pages
4. Select source branch (main/master)
5. Your site will be available at `https://[username].github.io/[repository-name]`

### Netlify

1. Sign up for a free Netlify account
2. Drag and drop the project folder to Netlify dashboard
3. Your site will be instantly deployed with a unique URL
4. Optional: Configure custom domain

### Vercel

1. Sign up for a free Vercel account
2. Import the project from GitHub or upload files
3. Deploy with one click
4. Optional: Configure custom domain

### Traditional Web Hosting

Simply upload all files to your web hosting provider's public_html or www directory using:
- FTP client (FileZilla, WinSCP)
- cPanel File Manager
- Hosting provider's upload interface
